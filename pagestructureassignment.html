Basic Information

Semantic Elements

Semantic Elements include Header, Nav, Section, Article, Aside, Main, and Footer.

|

Why use Semantic HTML

There are several reasons for using sematic HTML tags with your web page. It helps with organization of your page. It allows for better collaborations when multiple people are working on the same site. It better provides information about your page to browsers and search engines. Styling is even made easier when semantic HTML is consistantly used.

|

Alternatives

There are times when it might be necessary to use tags for containers that are not semantic HTML.

A div is a nonsemantic tag that can be specified by using class or id's to go along with it.
